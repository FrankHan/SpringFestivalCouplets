# 关于微信审核

这个程序我提交审核了4次：第一次审核通过；第二次由于页面有二维码审核未通过；第三次审核由于用户可以自定义内容，让我补充设计-笔记类目资质，未通过；第四次我直接把有可能出问题的地方都遮住，通过了审核，祝大家好运！
